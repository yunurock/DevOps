# Docker

Docker is an open source Platform that lets you package an application and all the it's dependieces in to a container
Ensuring it runs same way on any machine.

# **Hypervisor**

Hypervisor is a software that creates and manges Virtual machines (VM)
It allows multiple operating Systems to run on single Physical Machine, by sharing the hardware resources such as CPU, RAM, Storage
and Network.

# **Types Of Hypervisors**

**1. Bare metal Hypervisor**
**2. Hosted Hypervisor**

1. Bare metal Hypervisor runs directly on Physical Hardware. EXample VMWare ESXi and Xen

   **Advantages**
   1. Better Performance
   2. Higher Security
   3. Commonly Used in Data Centers and Cloud Environments.

2. Hosted Hypervisor runs on top of an exsting Operating system. Example Oracle and VMWare Workstation.

#**Virtualization**
Insted of using the Physical resources directly it allows Mutiple Virtual Machines to run on single Physical machine.

#**Containerzation**
It is a technology that packages an application along with all the it's dependencies libraries and configuration files into a container so it can run
consistently across different environments.

The container includes everything the application need to run.

Container share the Host as kernel instead of running a separate Operating System.

#**Virtualization V/s Containerzation**

1. Virtual Machines are dependent on Operating System.
2. Virtual Machine will take take to Start up and Shutdown.
3. Virtual Machine Size is very high and it is not flexiable to move from one env to another env.

1. Container are Independent of Operating System.
2. Containers will be very fast to start and shutdown.
3. Containers are very light we can easily move from one env to another env.
